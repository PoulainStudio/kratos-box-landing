# KratosBox - Landing Page Premium

Landing Page profesional y responsiva para KratosBox, gimnasio de Powerlifting y entrenamiento de fuerza.

## 🚀 Características

- ✅ **React.js** con componentes funcionales
- ✅ **CSS3 personalizado** - 100% personalizado, sin frameworks
- ✅ **Diseño Responsive** - Mobile-first, funcionando en todos los dispositivos
- ✅ **Animaciones Smooth** - Transiciones y efectos visuales de alto impacto
- ✅ **Modal Funcional** - Modal para "Método Kratos" con estado React
- ✅ **Scroll Suave** - Navegación fluida entre secciones
- ✅ **Formulario de Contacto** - Con validación básica
- ✅ **Paleta de Colores Premium** - Negro, Rojo Bordó y Blanco (tema Spartan)

## 🔧 Instalación

### Requisitos previos
- Node.js (v16 o superior)
- npm o yarn

### Pasos de instalación

1. **Clonar el repositorio** (o descargar los archivos)
   ```bash
   cd kratosbox-landing
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Iniciar servidor de desarrollo**
   ```bash
   npm run dev
   ```

4. **Construir para producción**
   ```bash
   npm run build
   ```

5. **Ver preview de la versión compilada**
   ```bash
   npm run preview
   ```
